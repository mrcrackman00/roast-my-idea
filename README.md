

markdown# 🔥 RoastMyIdea.ai

> "We'll destroy your startup idea. Then fix it."

An AI-powered tool that brutally roasts your startup idea, 
scores it across key metrics, and gives you actionable 
improvements to make it better.

## ✨ Features

- 💀 **Brutal AI Roast** — No sugarcoating. Pure honest feedback.
- 📊 **Damage Report** — Scores across Originality, Market Size, 
  Execution Difficulty & Competition
- 🚀 **Save Your Idea** — 3 specific actionable improvements
- ⚡ **Powered by Gemini AI** — Fast, accurate, brutal
- 📱 **Mobile Responsive** — Roast your ideas anywhere

## 🛠️ Tech Stack

- HTML, CSS, JavaScript
- Google Gemini API
- Vercel (Deployment)
